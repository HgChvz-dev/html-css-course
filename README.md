# HTML & CSS Course 🧱

Curso práctico y completo de HTML y CSS desde cero hasta nivel senior.  
Cada ejercicio cubre etiquetas, atributos y propiedades específicas con ejemplos reales.

## ¿Para quién es esto?

Para cualquiera que quiera aprender HTML y CSS de forma estructurada, progresiva y con ejercicios reales. Sin frameworks, sin atajos — base sólida primero.

## Estructura del curso

```
html-css-course/
├── html/
│   ├── 00-fundamentos/
│   ├── 01-texto-y-semantica/
│   └── ...
├── css/
│   ├── 00-fundamentos/
│   ├── 01-selectores/
│   └── ...
└── proyecto-final/
```

Cada carpeta contiene:
- `index.html` — ejercicio práctico
- `notas.md` — resumen de lo aprendido
