# 13 — HTML Integrator Project

## What was practiced

This project combined all HTML concepts learned across exercises 00–12.

| Area          | Tags / Attributes used                                                              |
| ------------- | ----------------------------------------------------------------------------------- |
| Structure     | `<header>`, `<main>`, `<aside>`, `<footer>`, `<section>`, `<article>`, `<nav>`      |
| Text          | `<hgroup>`, `<abbr>`, `<blockquote>`, `<cite>`, `<time>`, `<mark>`, `<small>`       |
| Lists         | `<ul>`, `<ol>`, `<li>`                                                              |
| Tables        | `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<th>`, `<td>`, `colspan`, `rowspan`    |
| Forms         | `<form>`, `<fieldset>`, `<legend>`, `<input>`, `<select>`, `<textarea>`, `<button>` |
| Multimedia    | `<figure>`, `<figcaption>`, `<picture>`, `<source>`, `<video>`, `<audio>`, `<img>`  |
| Special       | `<details>`, `<summary>`, `<meter>`, `<progress>`, `<iframe>`                       |
| SEO           | `<meta>`, Open Graph, Twitter Card, `<link rel="canonical">`                        |
| Accessibility | `aria-label`, `aria-live`, `role="alert"`, `tabindex`, `alt`                        |

## Rules confirmed

- `<aside>` goes inside `<main>`, not after it
- Every `<input>` needs `id` and `name` — including checkboxes
- `<address>` goes inside `<footer>`
- Internal nav links use `href="#section-id"` matching section `id`
- SEO meta tags all go inside `<head>`
