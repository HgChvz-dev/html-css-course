# 02 — Lists

## Tags learned

| Tag | Purpose |
|-----|---------|
| `<ul>` | Unordered list (bullets). |
| `<ol>` | Ordered list (numbers). |
| `<li>` | List item. Works inside `<ul>` and `<ol>`. |
| `<dl>` | Definition list. Groups all terms together. |
| `<dt>` | Term to define. Goes inside `<dl>`. |
| `<dd>` | Definition of the term. Goes after `<dt>`. |

## Key attributes

- `type` on `<ol>` — changes marker style: `1`, `A`, `a`, `I`, `i`
- `start` on `<ol>` — defines the starting number
- `reversed` on `<ol>` — counts down instead of up

## Rules learned

- `<dl>` groups all terms in one block — not one `<dl>` per term
- `<dt>` and `<dd>` replace `<p>` inside `<dl>`
- Lists can be nested — put a `<ul>` or `<ol>` inside a `<li>`
- `<li>` only goes inside `<ul>`, `<ol>` — never loose in the page

# 02 — Listas

## Etiquetas aprendidas

| Etiqueta | Para qué sirve |
|----------|---------------|
| `<ul>` | Lista desordenada (bullets). |
| `<ol>` | Lista ordenada (números). |
| `<li>` | Elemento de lista. Funciona dentro de `<ul>` y `<ol>`. |
| `<dl>` | Lista de definiciones. Agrupa todos los términos. |
| `<dt>` | Término a definir. Va dentro de `<dl>`. |
| `<dd>` | Definición del término. Va después de `<dt>`. |

## Atributos clave

- `type` en `<ol>` — cambia el estilo del marcador: `1`, `A`, `a`, `I`, `i`
- `start` en `<ol>` — define desde qué número empieza
- `reversed` en `<ol>` — cuenta al revés

## Reglas aprendidas

- `<dl>` agrupa todos los términos en un solo bloque — no uno por término
- `<dt>` y `<dd>` reemplazan a `<p>` dentro de `<dl>`
- Las listas se pueden anidar — poner un `<ul>` o `<ol>` dentro de un `<li>`
- `<li>` solo va dentro de `<ul>` u `<ol>` — nunca suelto en la página
